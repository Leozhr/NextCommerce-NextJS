# NextCommerce

Este projeto é uma loja virtual vibrante, transformando o comércio eletrônico com sua interface intuitiva e catálogo diversificado. Com tecnologia de ponta, oferece uma experiência de compra segura e agradável, levando o prazer do varejo direto para o conforto do lar do cliente.

## 💡 Construção
> O projeto foi desenvolvido em NextJS, React

<br/>

Este projeto foi desenvolvido com Next.js e React, seguindo a arquitetura Clean Code. A ferramenta utilizada para armazenar os produtos é o Sanity.io.

<br />

## ✨ Instruções de Uso

> Siga estas etapas para utilizar o projeto:

#### Clonar o Repositório:

Clone este repositório para o seu ambiente de desenvolvimento local usando o seguinte comando:

1 - ```git clone https://github.com/Leozhr/NextCommerce-NextJS.git```

2 - Abra a solução no ambiente de desenvolvimento de sua preferência (por exemplo, VSCode).

3 - ```Npm i```

<br />

## ⚙️ Iniciar Projeto

```
npm run dev
```
